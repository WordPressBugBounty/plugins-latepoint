## [5.2.0] - September 23, 2025

### Added
- Improved UI for admin dashboard
- Customers can log in using their phone number instead of email
- Import customers via CSV
- Ability for customers login with one-time code via SMS or email
- Support for short urls in SMS
- You can now disable email as a required field
- Guest booking, ability for customers to make a booking without creating an account
- Ability to attach files in emails through workflows
- Ability to attach an .isc calendar file for an appointment in workflows
- Link to customer information edit on summary panel
- Out of the box support for multiple languages (French, German, Spanish, Italian, Russian)

### Improved
- Improvements to how social login buttons look
